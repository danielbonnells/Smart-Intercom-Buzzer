This project is the code that runs on an ESP32 S2 mini to make a "smart" intercom in my old pre-war building. It detects a pulse from the door buzzing, wakes the device up, and sends a telegram notification to my phone, at which point I can decide to unlock the door for whoever buzzed.

## Materials used
- [ESP32 mini](https://www.amazon.com/HiLetgo-ESP32-S2FN4R2-ESP32-S2-Type-C-Connect/dp/B0B291LZ99/)
- [Relay Module](https://www.amazon.com/dp/B09G6H7JDT)
- 18650 Battery
- [Battery Charging Module](https://www.amazon.com/dp/B09JZFXD1X)
- Flexible wire, resistors, capacitor, diode
- 3D printer, soldering gun, hot glue gun
